
-----------------------------
   Command Line Cheat Sheet
-----------------------------
> ls               list files
  > -a               list all
> cd               change dir
> cd ..            up one dir
> mkdir              make dir
> rm               remove dir
  > -r        remove contents
> touch           create file
> open              open file
  > -a "app"          use app
  > .             current dir
> mv __ __    rename file/dir
> pwd       print working dir
